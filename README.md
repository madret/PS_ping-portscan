# swift_sweep
Lightweight ping sweep and port scan script written in Powershell. No additional powershell modules needed.

## User input:

### Specify the IP range
$subnet = "192.168.1"

### Specify the desired ports to scan
$ports = @(80, 443, 8080, 8443) # specify ports.
